4 Form controls #permalink

Form controls

Form controls allow users to enter information into a page.

4.1 Text input and area #permalink

Text input and area

Text inputs allow people to enter any combination of letters, numbers, or symbols of their choosing (unless otherwise restricted). Text input boxes can span single or multiple lines.

Default styling

.input:focus

Default button 

<fieldset>
  <form>
    <p>
      <label for="text_field">Text input label</label>
      <input type="text" id="text_field" autocomplete="off" placeholder="Placeholder">
    </p>
    <p>
      <label for="text_area">Text area label</label>
      <textarea id="text_area"></textarea>
    </p>
    <p>
      <label for="password">Password label</label>
      <input type="password" class="password" name="password" autocomplete="off">
    </p>
    <p>
      <label for="file">File Input</label>
      <input type="file" class="file" name="file">
    </p>
  </form>
</fieldset>
4.2 Checkboxes #permalink

Checkboxes

Checkboxes allow users to select one or more options from a visible list.

<fieldset>
  <form>
    <p>
      <input type="checkbox" class="checkbox" id="checkbox_1" name="checkboxes" value="checkbox_1">
      <label class="checkbox__label" for="checkbox_1">Checkbox 1</label>

      <input type="checkbox" class="checkbox" id="checkbox_2" name="checkboxes" value="checkbox_2">
      <label class="checkbox__label" for="checkbox_2">Checkbox 2</label>

      <input type="checkbox" class="checkbox" id="checkbox_3" name="checkboxes" value="checkbox_3">
      <label class="checkbox__label" for="checkbox_3">Checkbox 3</label>
    </p>
  </form>
</fieldset>
4.3 Radio buttons #permalink

Radio buttons

Radio buttons allow users to see all available choices at once and select exactly one option.

<fieldset>
  <form>
    <p>
      <input type="radio" class="radio" id="radio_1" name="radioboxes" value="radio_1">
      <label class="radio__label" for="radio_1">Radio 1</label>

      <input type="radio" class="radio" id="radio_2" name="radioboxes" value="radio_2">
      <label class="radio__label" for="radio_2">Radio 2</label>

      <input type="radio" class="radio" id="radio_3" name="radioboxes" value="radio_3">
      <label class="radio__label" for="radio_3">Radio 3</label>
    </p> 
  </form>
</fieldset>
4.4 Select / Dropdown #permalink

Select / Dropdown

A dropdown allows users to select one option from a list.

<fieldset>
  <form>
    <p>
      <label for="options">Select Element</label>
      <select name="options" id="options" class="select">
        <optgroup label="Option Group 1">
          <option value="1">Option 1</option>
          <option value="2">Option 2</option>
          <option value="3">Option 3</option>
        </optgroup>
        <optgroup label="Option Group 2">
          <option value="1">Option 1</option>
          <option value="2">Option 2</option>
          <option value="3">Option 3</option>
        </optgroup>
      </select>
    </p>
  </form>
</fieldset>
